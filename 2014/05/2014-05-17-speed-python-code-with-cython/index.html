先从一个例子来说，一个斐波那契数列数列例子说起：

在 CPython 下执行效果如下：

➜  cython  python try_c.py
429.003911972

换成 pypy 的话，效率会更高一些：

(pypy)➜  cython  pypy try_c.py
355.186796904

能不能有更快的效率呢？Python 的执行效率确实令人诟病，如果有 Python 的编写速度，C 的执行效率更好了（你说的是 Golang 嘛？）。这就是需要 Cython 出场了。
